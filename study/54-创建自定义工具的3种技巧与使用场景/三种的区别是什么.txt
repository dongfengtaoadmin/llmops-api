  详细说明

  1. @tool 装饰器 - 最简洁
  @tool("multiply_tool", return_direct=True)
  def multiply(a: int, b: int) -> int:
      return a * b
  - 自动从函数签名提取参数和描述
  - 自动处理同步函数，异步需额外配置
  - 适合快速原型开发

  2. StructuredTool.from_function - 平衡方案
  calculator = StructuredTool.from_function(
      func=multiply,
      coroutine=amultiply,  # 显式提供异步版本
      name="multiply_tool",
      return_direct=True,
  )
  - 可显式控制同步/异步函数
  - 适合需要同时支持同步和异步调用的场景

  3. BaseTool 子类 - 最灵活
  class MultiplyTool(BaseTool):
      name: str = "multiply_tool"
      args_schema: Type[BaseModel] = MultiplyInput

      def _run(self, *args, **kwargs):
          return kwargs.get("a") * kwargs.get("b")
  - 完全控制工具行为
  - 可重写 run、arun、invoke 等方法
  - 适合复杂业务逻辑、依赖注入、状态管理等场景
